# 🏥 HRPI - Sistema de Controle de Recepção

Sistema web para controle de acompanhantes e visitas do Hospital Regional de Palmeira dos Índios (HRPI), desenvolvido para a equipe de recepção.

## 📋 Funcionalidades

### 🎯 Principais
- ✅ **Controle de Entrada/Saída** - Registro completo de acompanhantes
- ✅ **Registro de Trocas** - Substituição de acompanhantes
- ✅ **Controle de Visitas** - Visitas temporárias com duração definida
- ✅ **Painel Dashboard** - Visão geral dos indicadores
- ✅ **Histórico Completo** - Todos os registros com filtros
- ✅ **Relatórios em PDF** - Diário, semanal, mensal e personalizado

### 👥 Gestão de Usuários
- ✅ **Múltiplos perfis** - Administrador, Supervisor e Recepcionista
- ✅ **Controle de permissões** - Acesso diferenciado por cargo
- ✅ **Primeiro acesso** - Troca obrigatória de senha
- ✅ **Reset de senha** - Recuperação para senha padrão

### 🎨 Personalização
- ✅ **Tema Claro/Escuro** - Alternância com um clique
- ✅ **Logo do Hospital** - Upload da imagem institucional
- ✅ **Fundo da Tela de Login** - Imagem de fundo personalizada
- ✅ **Totalmente Responsivo** - Desktop, tablet e mobile

## 🚀 Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| **HTML5** | Estrutura semântica |
| **CSS3** | Estilização com variáveis e temas |
| **JavaScript** | Lógica do sistema (ES6+) |
| **FireBase** | Armazenamento local dos dados |
| **jsPDF** | Geração de relatórios PDF |
| **Font Awesome** | Ícones profissionais |
