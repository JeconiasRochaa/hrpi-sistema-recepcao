# HRPI - Controle de Recepção

Sistema de controle de acompanhantes e visitas do Hospital Regional de Palmeira dos Índios.

## Como usar
1. Clone o repositório
2. Crie o arquivo `firebase-config.js` com as credenciais do Firebase
3. Abra o `index.html`

## Usuários padrão
| Usuário | Senha | Cargo |
|---------|-------|-------|
| admin | 123456 | Administrador |
| recepcao | 123456 | Recepcionista |
| social | 123456 | Serviço Social |

⚠️ No primeiro acesso, será solicitada a troca de senha.
